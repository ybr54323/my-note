## 怎么获取主机的公网ip
```shell script
curl icanhazip.com
```
icanhazip.com   是一个神奇的网址,访问时会返回主机的外网访问ip,类似的网址还有如下:
- http://ip.3322.net
- myip.dnsomatic.com
- ip.appspot.com
